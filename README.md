# Sysvar – ERP Web Platform

Enterprise Resource Planning (ERP) web platform developed for retail business management.

## Modules

- Purchase Orders
- Accounts Payable
- Inventory Control
- ERP Packs
- Financial Integration
- Audit Logs

## Features

- Automatic purchase order calculation
- Financial title generation
- ERP module integration
- Token Authentication
- Status workflow management

## Tech Stack

- Angular
- TypeScript
- REST API Integration
